# Gemini Studio | Shree Sai Hospital

A professional multispeciality hospital website for Shree Sai Hospital, featuring Dr. Ram Pawar. This project showcases specialized medical services, doctor profiles, and high-precision location information for the Dighi community.

## 🚀 Getting Started
1. `npm install` to install dependencies.
2. `npm start` to launch the development server.

## 🛠 Features
- **Authentic Infrastructure Showcase**: Displays the actual Shree Sai Hospital building photography for patient trust.
- **Blue Caduceus Branding**: Integrated official Caduceus logo with consistent medical blue styling across the site.
- **Modern UI**: Zinc-based aesthetic with high-contrast blue accents and professional typography.
- **Precise Ratings**: Custom star rating component displaying accurate 4.4/5 visual feedback with partial star filling.
- **Verified Location**: Integrated Google Maps for 100% location accuracy in Dighi.
- **Service Grid**: Modular display of 15+ medical specialities.
- **Doctor Dashboard**: A simulated administrative interface for managing appointments.
- **Optimized Navigation**: Clean menu structure focusing on core patient needs (Home, About, Services, Contact).
